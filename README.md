# 🍲 Comfort Cravings - Recipe Website

A cozy, fully responsive recipe website built as a web development class project. This site showcases 5 hearty comfort food recipes and features a modern, user-friendly design. 

## 🚀 Live Demo & Repository
* **Live Website:** [Comfort Cravings on GitHub Pages](https://parthmanish2008-hash.github.io/recipe_website/)
* **Project Repository:** [View Source Code](https://github.com/parthmanish2008-hash/recipe_website)

## ✨ Features
This project successfully fulfills all core assignment requirements and bonus objectives:

* **5 Distinct Recipe Pages:** Detailed pages for Red Sauce Pasta, Tomato Soup, Chocolate Cake, Butter Chicken, and Pancakes.
* **Fully Responsive Design:** Seamlessly adapts to desktop, tablet, and mobile screens using media queries.
* **Modern CSS Layouts:** Utilizes both CSS Flexbox (for navigation and search) and CSS Grid (for the recipe card gallery and About page).
* **Card-Based UI:** Featured recipes are displayed in a clean, interactive grid of cards.
* **CSS Animations:** Smooth fade-in and slide-up animations trigger when the page loads.
* **Categorization:** Visual, color-coded badges indicating recipe types (Veg, Non-Veg, Dessert).
* **UI Search Bar:** A styled, cohesive search bar interface built into the hero section.
* **Custom Typography & Color Palette:** A carefully selected earth-tone palette to create a warm, inviting aesthetic.

## 🛠️ Technologies Used
* **HTML5:** Semantic document structure.
* **CSS3:** Custom styling, animations, Grid, and Flexbox layouts.
* *(Strictly no JavaScript per assignment requirements).*
